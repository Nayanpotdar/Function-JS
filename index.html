Q7 Write a function that accepts parameter n and returns factorial of n

             function factorial(n) {
                if (n === 0 || n === 1) {
                  return 1;
                } else {
                  return n * factorial(n - 1);
                }
              }
              
              console.log(factorial(5));

Output: 120 (5! = 5 * 4 * 3 * 2 * 1 = 120)
